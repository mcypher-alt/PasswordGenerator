# Password Generator (Node.js CLI)

Генератор случайных паролей с настройками прямо в командной строке.

## Функционал

- Задание длины пароля
- Выбор использования заглавных букв
- Выбор использования цифр
- Защита от неверного ввода (рекурсивный запрос)

## Установка и запуск

```bash
git clone https://github.com/mcypher-alt/PasswordGenerator.git
cd PasswordGenerator
npm install readline-sync
node main.js
